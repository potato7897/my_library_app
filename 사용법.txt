파이썬 설치 후
코드 파일 zip으로 받은 다음 압축 해제 후 해당 폴더에서 마우스 오른쪽 클릭으로 터미널 열기

pip install -r requirements.txt
명령어 실행

아래 명령어는 코드 실행

python main.py

파일 폴더에서 rj숫자열 추출


python crawler.py

추출된 숫자열을 바탕으로 웹 크롤링 진행


python library_app.py

db 바탕으로 라이브러리를 만들어서 열어봄

한번 main.py와 crawler를 해두면 파일이 추가됬을때만 한번씩 돌리고
그냥 라이브러리를 열려면 
python library_app.py만 하면됨

모르겠는거 있으면 긁어서 ai한테 물어보셈
